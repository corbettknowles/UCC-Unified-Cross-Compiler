# UCC - Unified Cross Compiler

## Description
The Unified Cross Compiler is built using a modified LLVM base 
as the foundation along with other open source components and 
personal source code.

## Status
This project is still in the early stages of development. 
The first build will be available within 6 months to 1 year.

## Target Architectures
UCC currently targets 22 architectures:

- ARM32
- ARM64 / AArch64
- AVR
- i386
- i686
- IA32
- IA64
- MIPS32
- MIPS64
- PowerPC (PPC32)
- PowerPC 64 (PPC64)
- PowerPC 64 Little Endian (PPC64LE)
- RISC-V 32 (RISCV32)
- RISC-V 64 (RISCV64)
- S390X
- SH2
- SH3
- SH4
- SPARC32
- SPARC64
- x86 (x86_32)
- x86_64
